# CiteFlow 📚

CiteFlow is a full-stack MERN application for managing and organizing research papers using structured metadata, topic-based categorization, and an interactive dashboard.

---

## 🚀 Features

* 📄 Add, edit, and manage research papers with metadata (title, authors, journal, year, DOI)
* 🗂️ Organize papers by topics
* ⭐ Mark papers as favorites for quick access
* 🔍 Search papers by title and author
* ⬆️ Sort papers based on different criteria (year, title)
* 📊 Interactive dashboard with real-time statistics

## 📦 Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/citeflow.git
cd citeflow
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run Backend

```bash
cd backend
npm run dev
```

### 4. Run Frontend

```bash
cd frontend
npm run dev
```
